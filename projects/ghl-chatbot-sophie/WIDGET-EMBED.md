# Sophie Widget Embed

## Embed Code (any website)

Paste before the closing `</body>` tag:

```html
<script src="https://widgets.leadconnectorhq.com/loader.js" 
  data-resources-url="https://widgets.leadconnectorhq.com/chat-widget/loader.js" 
  data-widget-id="YOUR_WIDGET_ID">
</script>
```

## WordPress

Install the **LeadConnector** plugin and enter the Widget ID.

## Notes

- Chat type: Live Chat (sticky, floats on page corner)
- Platform: GoHighLevel (white-label)
