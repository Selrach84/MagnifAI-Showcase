# Engagement Plan — 12-Week Build

## Objective
Transform [Roofing Company] into a tech-enabled media company. Build the complete technical foundation — GoHighLevel CRM, AI estimator, automated lead gen, phone system — then package as a service for other contractors.

## Phases

### Phase 1a: Foundation (Weeks 1-3)
| Priority | Component | Details |
|----------|-----------|---------|
| 🥇 | GoHighLevel Setup | Full CRM: lead capture forms, pipeline stages, automated follow-up sequences, SMS/email nurture |
| 🥇 | Website | Modern, mobile-first, optimized for storm-damage leads |
| 🥇 | Google Ads | Targeted campaigns geotargeted to storm zones |

**Deliverable:** Leads visible within 2-3 weeks.

### Phase 1b: AI Automation (Weeks 4-6)
| Priority | Component | Details |
|----------|-----------|---------|
| 🥇 | AI Claims Triage | Auto-review insurance claims in 5 min vs 30-40 min |
| 🥇 | Lead Qualification Bot | Website + SMS bot for capture, qualify, book |
| 🥈 | Automated Follow-Ups | Post-estimate, post-job, referral sequences |

**Deliverable:** Operational workflow 3x faster.

### Phase 1c: Intelligence & Scale (Weeks 7-9)
| Priority | Component | Details |
|----------|-----------|---------|
| 🥇 | Competitive Intelligence | Monitor competitors for market signals |
| 🥇 | Dashboard | Real-time view: leads, pipeline, ad ROI |
| 🥈 | Referral Partner Network | Automated partner drip campaigns |

### Phase 1d: Polish & Showcase (Weeks 10-12)
| Priority | Component | Details |
|----------|-----------|---------|
| 🥇 | Case Study Package | Before/after metrics |
| 🥇 | Demo System | Living demo for prospects |
| 🥈 | Service Menu & Pricing | Defined packages |

## Top Priorities
1. AI Estimator — #1 technical priority
2. CRM migration — replace legacy tools as one-stop platform
3. Lead generation — capitalize on seasonal demand window
4. Phone system — AI-backed 24/7 coverage
5. Team building — dedicated delivery team
