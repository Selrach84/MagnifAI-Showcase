## Activation rate formula
```
Activation Rate = (Features Active / Features Licensed) × 100
```

## Common findings
- CRM 80% licensed but 20% activated — massive untapped investment
- AI agents exist but backend integration missing
- Calendars exist but not connected to lead capture
- Workflows created but never triggered
- No data flowing between website → CRM → calendar → email

## Opportunity areas
- Activate dormant features (highest ROI per hour)
- Connect isolated systems (website → CRM → email)
- Build automated follow-up sequences
- Implement lead scoring and routing
- Set up reporting dashboards
